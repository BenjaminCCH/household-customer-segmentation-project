# Household Customer Segmentation Report 

## Overview
This project applies multiple unsupervised machine learning techniques to segment customers based on demographic and purchasing behaviour data. The goal is to identify meaningful customer groups that enable more targeted and effective marketing strategies.

## Tools
- Python
- Jupyter Notebook

## Data Source
Dunnhumby – The Complete Journey (Kaggle)  
Retrieved from: https://www.kaggle.com/datasets/frtgnn/dunnhumby-the-complete-journey

Detailed information about this dataset can be obtained from GitHub (CSC465-visualization-project/Dataset/dunnhumby).
Retrieved from: https://github.com/Lanbig/CSC465-visualization-project/blob/master/Dataset/dunnhumby%20-%20The%20Complete%20Journey%20User%20Guide.pdf

## Steps to Run the Project
1. Download all 8 CSV files (2019–2023) from the `raw datasets` folder.  
2. Download all the `.ipynb` files from the `scripts` folder.  
3. Open all scripts in Jupyter Notebook and update the dataset file paths.  
4. Run the scripts in order from 1 to 4.  

If you download the cleaned dataset (`customer_segmentation_after_preprocessing.csv`) from the `clean dataset` folder, you only need to run script 4.
